== Description ==

WooCommerce based plugin that replaces all images that are coming with WooCommerce by default with div element, thus avoiding any image resize conflicts.
Note: You should have WooCommerce plugin installed to use it.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
